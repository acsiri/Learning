# Contributing to Learning Repository

Thank you for your interest in contributing to this learning project!

## How to Contribute

### 1. Fork and Clone
```bash
git clone https://github.com/acsiri/Learning.git
cd Learning
```

### 2. Create a Branch
```bash
git checkout -b feature/your-feature-name
```

### 3. Make Your Changes
- Keep changes focused and well-documented
- Add comments explaining complex logic
- Test your changes thoroughly

### 4. Commit Your Changes
```bash
git add .
git commit -m "Add descriptive commit message"
```

### 5. Push and Create a Pull Request
```bash
git push origin feature/your-feature-name
```

## Code Guidelines

- Follow PEP 8 style guide for Python code
- Use meaningful variable and function names
- Add docstrings to functions and classes
- Keep functions focused and modular

## Security

⚠️ **Important:** Never commit your API keys or sensitive credentials
- Use environment variables for API keys
- Add `.env` to `.gitignore`
- Reference `README.md` for proper setup

## Questions?

Feel free to open an issue for questions or discussions about the project.

Happy coding! 🚀
